Code Violations

What is Code violation

=> Municipalities ( পৌরসভা ) have rules and regulations that must be followed by all properties. Should a property not be in accordance with these regulations, a code violation can be issued. These are often for overgrown/tall grass, unsecured pool, excessive debris, infestation, etc.

Violations

Common Violations:

  1. Exterior debris
  2. Health or trip hazards
  3. Overgrown grass
  4. Overgrown shrubs/trees
  5. No “means of egress” (refers to the ability to exit the structure, in the event of an emergency, such as a fire)

Can be issued by:

  1. County/City
  2. HOA
